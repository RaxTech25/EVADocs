4. Configuration Steps

  1. Search for config.json file in the EVA directory (/home/pi/EVA )
    • Right Click on config.json file and click on View/Edit.
    • Change the tcp IP address as Device IP or required IP
    • Save and close the file
    • In Filezilla Click on YES to reflect the changes in files.

  2. Search for host.json file in the EVA directory (/home/pi/EVA )
    • Right Click on host.json file and click on View/Edit.
    • Change the IP address as Device IP or required IP
    • Save and close the file
    • In Filezilla Click on YES to reflect the changes in files.

  3. After Changing the Config.json and Host.json files, restart the EVA Device to apply the changes.

5. Network Setup

  1. Check the EVA Device Ethernet connectivity with the router or local PC based on the user environment.
  2. If the Device is connected locally with the device IP series as 1, the local IP should be in the same 1 series.
    (Eg: Local - 192.168.1.220, Device IP - 192.168.1.219).
  3. If the Device is connected remotely, ensure proper port forwarding is set up in your router environment.
  4. Use ping to check the Device connectivity.
  5. Use telnet to check Device Communication via ports 22, 6000, and 3000.

7. Custom Audio Generator

  • Visit the site to convert text to audio ‚Üí üîó Link
  • Input the text you want to convert to speech.
  • For Different language use Google Translate to generate text.
  • Adjust the text accordingly based on audio.
  • Select the desired voice and language options.
  • Click on the "Generate" button to generate the audio file.
  • Download the generated audio file to your computer.
  • Transfer the audio file to the EVA device using Filezilla.
  • Place the audio file in the appropriate directory on the EVA device. (/home/pi/EVA/Audios)
    • Audio file name should match the existing file name to play the audio.
  • Restart the EVA Device to apply the changes.
  • If the Audio volume is low, check the volume settings on the EVA device.
      • If Audio is low, try to boost the audio online üîó Link
    1. Upload the audio file which need to boosted.
    2. Select Podcast Standard instead of Normalize Audio.
    3. Increase the volume to 200%.
    4. Select the mp3 format for the audio file to download.
    5. Click on Export Audio.
    6. Replace the original audio file on the EVA device with the new one.
